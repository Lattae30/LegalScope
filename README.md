# Requirement to run backend API:
- Install express:
  ```bash
  npm i express
  ```

- Install MongoDB:
  ```bash
  npm install MongoDB
  ```

- Install Mongoose for accessing mongoDB from Node.js:
  ```bash
  npm install mongoose
  ```
  
- Running Project:
  ```bash
  npm run serve
  ```

- Running Project Using Nodemon:
  ```bash
  nodemon index.js
  ```
